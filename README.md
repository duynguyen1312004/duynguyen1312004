### 🚀 Featured Projects

- **💬 [Real-Time Chat Application](https://github.com/duynguyen1312004/realtime-chat-app)**  
  A real-time chat application built with **Node.js**, **Express**, and **Socket.IO**, following the **MVC architecture** with Controller, Service, and Store layers. Features include real-time messaging, online users, typing indicators, and a scalable project structure.

- **🫀 [TinyML ECG Monitor](https://github.com/duynguyen1312004/TinyML-ECG-Monitor-Project)**  
  An AI-powered real-time ECG monitoring system running **TinyML** directly on an **ESP32**. The project combines Embedded Systems, IoT, MQTT, and Edge AI for arrhythmia detection.

- **🎬 [CineGo - Movie Ticket App](https://github.com/duynguyen1312004/CineGo)**  
  An Android movie ticket booking application integrated with **Firebase** and **Google Gemini AI**, providing movie recommendations and an intelligent virtual assistant.

- **👟 [Shoe Store Management System](https://github.com/duynguyen1312004/shoe-store-ecommerce-backend)*  
  A management system for shoe retail operations, including product management, customer management, inventory tracking, and sales processing, developed to practice software engineering principles and database design.
